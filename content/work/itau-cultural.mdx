Projeto de design para o site do Itaú Cultural, que foi totalmente remodelado tendo como princípios maior flexibilidade para conteúdo, interação com os usuários e integração com as redes sociais. Um novo portal foi criado a partir de um longo estudo de arquitetura de informação, o que permitiu que o conteúdo tivesse prioridade dentro de uma estrutura modular e escalonável.
